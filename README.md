# tools

A collection of utility scripts for media management and automation, running on a Raspberry Pi (RK3588).

## Contents

| Tool | Description |
|---|---|
| [video_subtitle](video_subtitle/README.md) | Burn, extract, or translate subtitle tracks from MKV video files using ffmpeg and Google Translate |

## Requirements

- Python 3.10+
- Raspberry Pi OS (or any Debian-based Linux)

Each tool has its own README with detailed requirements and usage instructions.
